Run the /dfy-weekly skill for Joy for the current week.

The week runs Friday → Thursday. Today is Friday, so the target week is the 7 days ending yesterday (Thursday): i.e. last Friday through yesterday Thursday. Compute the exact startDate (last Friday) and endDate (yesterday, Thursday) yourself from today's date.

Follow skills/dfy-weekly/SKILL.md exactly:
- Read AVD_TICKET_API_KEY from CSL/.env
- Fetch tickets from https://avada-ts-a9cb0.web.app/api/external/tickets/by-date with appName="JOY Loyalty", startDate, endDate
- Fetch tags from https://avada-ts-a9cb0.web.app/api/external/tags
- Identify DFY tickets (subject starts with [DFY]), determine CS creator (members[].isCreate), apply the open-only / sale_request / liz-no-tag filters
- Build the 3-line Overview block: Số ticket (filtered open set), Adopted (DFY-adopted), Ticket không có tag adopted (= Số ticket − Adopted)
- Build per-CS breakdown tables: Date | Ticket | Store | Tasks | Tags  (NO Point column, NO Status column)
- Write the markdown to a temp file /tmp/joy-dfy-{YYYY-W##}.md (ISO week of the Thursday end date). The H1 first line must be "# Joy DFY - {start} → {end}".

Then publish to Notion as a NEW sub-page (newest on top — do NOT save into the repo, do NOT commit to git):
