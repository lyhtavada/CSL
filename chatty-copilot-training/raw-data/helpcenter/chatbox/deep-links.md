### [hashtag](#what-is-deep-links) What is deep links?

Deep links allow you to create a special URL. When this URL is clicked, it automatically opens the Chatty chatbox on your Shopify store page.

You can add this link to a button or a hyperlink text that you want to use as a trigger.

### [hashtag](#how-does-it-work) How does it work?

1. Go to "Chatbox" -> Click **Advanced**
2. Copy deep link. If you want to open tab "Order tracking" in chatbox, copy deep link of Order tracking.
3. Add to your URL.
4. Add link to a button or anything on your website to use as a trigger.

For example, you can add a new button in your navigation.
