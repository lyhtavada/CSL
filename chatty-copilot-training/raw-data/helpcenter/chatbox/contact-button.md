**Who can use this feature?**

- This feature is available for all users

You can add contact button on your site to provide instant support and improve customer satisfaction.

Contact button works on Product and Cart page.

1. Go to Chatbox
2. Click **Turn on** Contact button
3. Click **Go to theme** to add contact button to your theme
4. In theme editor, select position you want to add contact button -> Click **Add block**
5. Select **Contact button** by Chatty
6. Set up contact button on the right panel.

   circle-info

   You can only set up contact button with active contact methods. Active contact methods are those you set up in [chatbox blocks.](/chatbox/chatbox-settings#customize-chatbox)

### [hashtag](#remove-contact-button) Remove contact button

To remove contact button, go to theme editor, click on the Contact button block and click **Remove block.**

circle-info

If you delete contact method in Chatty, contact button won't be deleted. Go to theme editor and delete that block.
