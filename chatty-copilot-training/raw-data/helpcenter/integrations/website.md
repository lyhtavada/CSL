### [hashtag](#what-is-website-integration) What is website integration?

Website integration allows you to add Chatty to any website you own, even if it's not hosted on Shopify.

This means your customers can get instant answers whether they're on your blog, landing page, or any custom website. And you can manage all those conversations in Chatty only.

### [hashtag](#why-use-website-integration) Why use website integration

- Capture sales opportunities across all your websites
- Provide consistent 24/7 support everywhere your brand appears
- Manage all customer conversations from one unified inbox

### [hashtag](#how-to-add-chatty-to-your-website) How to add Chatty to your website

#### [hashtag](#before-you-start) Before you start:

Make sure you have:

- Access to your website's HTML files (or content management system)
- Basic ability to add code snippets to your website (or access to someone who can)

You don't need technical expertise. Just the ability to copy and paste code into your website.

1

#### [hashtag](#get-your-widget-code) Get your widget code

1. From your Chatty admin, go to Settings
2. Click **Integrations**
3. Click **Manage** in Website integration
4. Your unique code snippet will be automatically generated

2

#### [hashtag](#copy-the-code-snippet) Copy the code snippet

1. Select type of code: `HTML` or `Liquid`
2. Click **Copy**

3

#### [hashtag](#add-code-to-your-website) Add code to your website

1. Open your website's HTML file
2. Locate the `<body>` section
3. Paste the code snippet just before the closing `</body>` tag
4. Save your changes

4

After you've added the code, the Chatty chat widget will appear on your website within a few minutes.

### [hashtag](#how-does-it-work) How does it work?

When customers visit your other websites:

- They'll see the same Chatty chat widget as on your Shopify store
- AI assistant responds to their questions instantly using your trained data
- All conversations from your external websites flow into the same inbox

### [hashtag](#faqs) FAQs

chevron-rightCan I add Chatty to multiple websites?[hashtag](#can-i-add-chatty-to-multiple-websites)

Yes, you can use the same code snippet on as many websites as you want. All conversations will flow into your single Chatty inbox.

chevron-rightWill this slow down my website?[hashtag](#will-this-slow-down-my-website)

No, the Chatty widget loads asynchronously, which means it won't affect your website's loading speed.

chevron-rightCan I customize how the widget looks on different sites?[hashtag](#can-i-customize-how-the-widget-looks-on-different-sites)

The widget will use your default Chatty settings.

chevron-rightWhat if I need to remove Chatty from my website?[hashtag](#what-if-i-need-to-remove-chatty-from-my-website)

Simply delete the code snippet from your website's HTML.

chevron-rightDoes this work with website builders like WordPress or Wix?[hashtag](#does-this-work-with-website-builders-like-wordpress-or-wix)

Yes, as long as you can add custom HTML code to your website, the integration will work. Most website builders have a custom code section where you can paste the snippet.
