### [hashtag](#import-faqs) Import FAQs

You can import many questions and answers to Chatty by CVS file.

1

#### [hashtag](#get-the-sample-file-to-import-faqs) Get the sample file to import FAQs

Go to **FAQs** → Select **More actions** → Click **Import FAQs** → Download a sample CVS

This is samp file to import FAQs

2

#### [hashtag](#enter-your-questions-and-answers-in-cvs-file) Enter your questions and answers in CVS file

It is required to fill in all information (keep hearders).

In sample file, there are 5 fields:

- Question
- Answer
- Category: Category name that question belongs to
- Published question: TRUE = published question, FALSE = unpublished question
- Featured question: TRUE = featured question, FALSE = not featured question

circle-info

File size must be less than 1MB

3

#### [hashtag](#save-and-upload-file) Save and upload file

Click Import FAQs -> Drop CVS file and select your option for all FAQs

- Overwrite FAQs that have the same questions
- Published all new categories

### [hashtag](#export-faqs) Export FAQs

You can also export FAQs in CVS file.

Go to **FAQs** → Select **More actions** → Click **Export FAQs** → Select which FAQs you want to export → Click Export

In FAQs file, there are 7 fields:

- Question
- Answer
- Category: Category name that question belongs to
- Published question: TRUE = published question, FALSE = unpublished question
- Featured question: TRUE = featured question, FALSE = not featured question
- Published category: TRUE = published category, FALSE = unpublished category
- Featured category: TRUE = featured category, FALSE = not featured category
