W**ho can use this feature?**

- This feature is available for all users

In FAQs analytics, you can check total views of FAQs

View counts each time a user clicks to expand and see the answer to a question on the FAQs page, in an FAQs block, or within the chatbox.

- Go to **FAQs** -> Check total views of each questions
- Go to **Dashboard**, you can check overview of FAQs view and filter by time range.
