W**ho can use this feature?**

- This feature is available for all users

Translation is a feature helps translate all content in chatbox into your language automatically. It saves your time and effort on manual translations for FAQ labels.

### [hashtag](#how-does-translation-work) How does Translation work?

Your chatbox will be shown in [Default language](/others/translation#default-language) to all users in default.

If you publish more languages, those will be available for users who use that language on your website.

#### [hashtag](#set-up-languages-of-your-website) Set up languages of your website

Before publishing a new language, make sure your website is available in that language.

In Shopify Admin, click **Settings** -> **Language** and set up your language.

### [hashtag](#default-language) Default language

Default language is available for all users. English will be set as default language when you first using app.

If user's language is not available in your translation, chatbox will be shown in default language.

circle-info

With users who downloaded app before April 23, 2024: If you already set a language as a primary language, that language will be set as default

#### [hashtag](#change-default-language) Change default language

Only published language can be updated as default language.

1. Go to translations
2. In published language list, click  -> click **Change default**
3. Click **Save**

### [hashtag](#published-language) Published language

Published languages will be available for visitors of your website using those languages.

#### [hashtag](#publish-a-language) Publish a language

1. Click **Add language**
2. Newly added language will be listed in the "Unpublished language". It will be auto-translated with the latest content.
   Click **Edit** if you wantedit your translation. If you don't want to edit translation, skip to step 5 to **Publish language**.
3. You can edit translation of Chatbox, FAQs page and all your FAQs.
   Click **Auto-translate** to translate all content, or you can edit content yourself.

   In FAQs translation, click  to edit categories and questions.

### [hashtag](#update-translation) Update translation

When you add new categories and questions, translation won't be updated automatically.

We'll show a banner to help you update translation. In translation, all languages which are not updated with new FAQs have `Outdated` status.

To update translation, click **Edit.**
