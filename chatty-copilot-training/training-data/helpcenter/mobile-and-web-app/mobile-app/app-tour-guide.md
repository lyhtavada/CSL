Q: What is App Tour Guide in Chatty?
Q: How do I set up App Tour Guide?
Q: How do I use App Tour Guide on mobile/web?
A: In Chatty mobile app, you can:

- Reply to all conversations
- Set up main settings: [general settings](/others/general-settings), [notifications](/others/notifications), [online hours](/others/online-hours) and [quick replies](/live-chat/quick-replies)

In conversation list, you can check all convesations and reply with all conversations. You can still get access to all functions in chat: send files, quick replies, [and more](/live-chat/inbox/chat-zone).

In **Settings**, you can set up all settings in app.
