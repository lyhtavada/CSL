Q: What is 🔔 Web push notifications in Chatty?
Q: How does 🔔 Web push notifications work?
Q: Where can I find 🔔 Web push notifications settings?
A: Setting notifications for messages helps you take care of all customers. No missing messages.

#### There are 3 types of notification

- Web push notifications: Send push notifications to your web browser. Make sure that your allow notifications on your device.

  - Sound notifications (advanced): Web browser will play sound for new messages (only from end users' web browser, when you send them messages)
- Email notifications: Send emails for new messages
- Mobile notfication: Download Chatty mobile app for notifications on mobiles

### 🔔 Web push notifications

You need to turn on notifications

#### Turn on Notifications in app

Go to Notifications -> In **Notify me when**, select "Web" for options you want to get notifications:

- There is a conversation assigned to me
- There is a new message from unassigned conversations
- There is a new message from conversations assigned to me
- There is a new message from AI assistant conversations
- There are no replies in 15 minutes from conversations assigned to me

#### [Turn on Notifications on web browser](/others/notifications#turn-on-notifications-on-web-browser-1)

Currently, we support web push notifications on these web browsers: [Chrome](/others/notifications#chrome), [Microsoft Edge](/others/notifications#microsoft-edge), [Firefox](/others/notifications#firefox), [Opera](/others/notifications#opera), and Samsung Internet.

Read detailed instructions for each web browser below.

If you are using different web browser which is not in the list, contact our support team through live chat in app.

#### Turn on Notifications on your device

You need to enable notifications for that web browser on your device (Windows, MacOS, Mobile) so that you can receive push notifications.

#### Windows

1. Go to Settings -> System -> Notifications (or Notifications & actions)
2. In "Get notifications from these senders", find your web browser
3. Turn on notifications

#### MacOS

1. Click Apple menu  -> Click Notifications  in the sidebar
2. In Application Notifications, find your web browser
3. Turn on notifications

#### Mobile

To get notifications for new messages on mobiles, [download Chatty app.](/mobile-and-web-app/mobile-app)

#### Turn on Notifications on web browser

#### Chrome

In **Web push,** click **Manage**

You might see a new window is opened. Click

Turn on **Notifications**

Get back to Chatty and click to **Save**.

Click **Send test** to check.

#### Microsoft Edge

In **Web push,** click **Manage**

You might see a new window is opened. Turn on **Notifications**.

Refresh page. Click to **Save**.

Click **Send test** to check if notification works.
