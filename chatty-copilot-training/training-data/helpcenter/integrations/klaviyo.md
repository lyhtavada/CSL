Q: What is Klaviyo Email Marketing & SMS in Chatty?
Q: How do I set up Klaviyo Email Marketing & SMS?
Q: How do I connect Klaviyo Email Marketing & SMS with Chatty?
A: ### What is Klaviyo Email Marketing & SMS?

Klaviyo is an email & SMS marketing platform that helps you send targeted emails and SMS messages to your customers.

It lets you create personalized marketing campaigns based on customer behavior and purchase history.

Learn more about [Klaviyo](https://apps.shopify.com/klaviyo-email-marketing?utm_source=chatty)

### Why integrate Klaviyo?

This intergration helps sync customer data and chat interactions between Chatty and Klaviyo.

- Keep customer data consistent across platforms
- Build smarter customer segments based on chat interactions
- Track chat interactions for targeted email campaigns

### How to integrate Klaviyo

#### Go to Integrations

Get access to Chatty admin → Click Settings from the navigation → Go to Integrations

#### Go to Klaviyo

In "Klaviyo", click **Manage**

#### Install Klaviyo (step 1)

Click **Go to Klaviyo** and install app. If you have already installed Joy, skip this step.

#### Enter Klaviyo API key

You’ll need your Klaviyo API key to connect your account. The API key helps sync your customer data securely between Chatty and Klaviyo.

[How to get Klaviyo API key](/integrations/klaviyo#how-to-get-klaviyo-api-key)

#### Click Connect

By clicking **Connect**, this integration syncs the following customer data to Klaviyo:

- Contact information (name, email, phone, location, type)
- Tags
- Number of conversations
- Last chat timestamps (when last chat ends and start)

#### How to get Klaviyo API key

#### Log in to Klaviyo

#### Go to Profile settings

Click on your profile on the bottom left corner and click **Settings**

#### Create API key for Chatty

- Go to **API keys**
- Click **Create Private API key**
- Name your API keys (For example: Chatty integration)
- Select Access Level: Custom key
- In API scopes, select **Read/Write Access** for: List and Profiles

#### Click Create

You created API key successfully for Chatty. Copy and paste it into Chatty settings to integrate apps.
