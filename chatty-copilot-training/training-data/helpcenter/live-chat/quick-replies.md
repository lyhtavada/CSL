Q: What is Create quick replies in Chatty?
Q: How do I set up Create quick replies?
Q: How do I use Create quick replies in live chat?
A: Quick replies help users set up messages which are common for merchants to answer customers.

### Create quick replies

1. Go to **Settings** -> In Quick replies, click **Manage**
2. Turn on Quick replies to enable quick replies in chat
3. Click Create quick reply
4. Enter shortcut, category and your response.
   You can enter category name and click Add to add new category.
5. Click **Save**
6. You can edit or delete quick replies.

### Use quick replies in chat

In conversation, click icon  or type  `/`  to get access to quick replies. Then simply click to select your quick replies.

In chat, click icon  to add new quick replies, or go to quick replies settings by clicking icon .
