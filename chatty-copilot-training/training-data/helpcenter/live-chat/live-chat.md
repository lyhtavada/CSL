Q: What is live chat in Chatty?
Q: How do I set up live chat?
A: ### What is live chat?

Live chat helps your customers contact to you while visiting your store. Also you can respond right away and [manage chats](/live-chat/inbox) more effectively.

This how live chat works on your store front

### Set up live chat

1. Make sure you turn on Chatbox. (Go to Chatbox -> In "General", click **Turn on** chatbox)
2. In "Blocks", turn on **Live chat**
3. Click Edit to set up pre-chat form.
   Pre-chat form helps you collect customer's info before chat starts.

   To check preview of pre-chat form, click **Chat now** in preview and enter your first message.

   "Required info before chat starts" : Helps you set up info you want to collect from customers. Email is required. You can select Name and Phone more.
   "Chat as anonymous": If you allow chat as anonymous, users don't need to enter info to start chat. In pre-chat form, there's an option for users to "Chat with us as anonymous".
4. Click **Save**
5. Go to your store front to check how chatbox looks like on your store front. Click **Chat now** in chatbox to start a test conversation.

You can set up online/offline status in [online hours.](/others/online-hours)
