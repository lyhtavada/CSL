# FAQ Builder

## Categories
Organize questions by topic. Two defaults: "Order & Shipping" and "Exchange & Return".

- Create: FAQs > Add new > Add category. Set title, icon, position, status (published/draft), featured status.
- Featured: If 2 categories featured, higher position shows first.
- **Uncategorized**: Default category, cannot be deleted. When you delete a category, its FAQs auto-move here.
- Manage: Edit, delete, reorder by drag, toggle featured.

## Questions
- Create: FAQs > Add new > Add FAQ. Enter question + answer, set status, assign category (required), optionally mark featured.
- Manage: Hover to edit/delete, drag to reorder, star for featured.
- **Import**: CSV with 5 fields (Question, Answer, Category, Published, Featured). Max 1MB. Option to overwrite duplicates, auto-publish new categories.
- **Export**: CSV with 7 fields including category and publication details.

## FAQs Page
Dedicated page on storefront.

Setup:
1. Enable "Display FAQs page"
2. Configure URL
3. Customize layout + colors (use contrast checker for readability)
4. Set header (heading + description)
5. Configure contact options
6. Advanced: branding removal, custom CSS

Add to navigation: Shopify > Content > Menus > add link to main menu or footer.

## FAQs Block
Custom FAQ sections for specific pages/products.

- Create multiple blocks tailored to products/collections
- Display conditions: all pages, specific product pages, or all products in a collection
- Customize: heading size, alignment, corner radius, colors
- Setup: configure > save > copy Block ID > add to theme editor
- Unlimited blocks, reuse FAQs across blocks

## FAQs Analytics
Tracks views (each answer expansion = 1 view) across FAQ page, blocks, and chatbox. View per-question counts in FAQs section, or overview on Dashboard with time range filter.

## Translation
19 supported languages + Custom option. Default is English.

- New languages start Unpublished with auto-translation
- Can manually edit translations for chatbox, FAQ pages, individual FAQs
- Must add/publish language in Shopify Admin > Settings > Languages first
- Adding new FAQ categories/questions doesn't auto-update translations — shows "Outdated" status
