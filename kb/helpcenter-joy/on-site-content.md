# On-Site Content

Embedded loyalty touchpoints on the storefront. Sections available for the loyalty landing page:

1. **Hero Banner**: Top section with branding
2. **How It Works**: Explains program mechanics
3. **Ways to Earn**: Shows earning programs
4. **Ways to Redeem**: Shows redeeming options
5. **Rewards Activity**: Customer's reward history
6. **Referral Program**: Referral section
7. **VIP Tier Benefits**: Tier comparison
8. **Loyalty Program FAQs**: Program-specific FAQ
9. **Coming Soon Teaser**: For upcoming features

## Other Touchpoints
- **Loyalty Widget (Unified Widget)**: Floating widget — points balance, available rewards
- **Account Page**: Loyalty dashboard, redemption, referral management, wallet pass
- **Product Page**: Point calculator, product referral, member exclusive deal
- **Cart Drawer**: Redeem points in cart
- **Checkout Page**: Quick redeem, available rewards, all discounts, referral block
- **Thank You Page**: Sign-up block, referral block, reward celebration

## Reviews tab in Unified Widget (Judge.me)
With Judge.me connected, the Unified Widget shows a **"My Reviews" tab** — customers view reviews they've written and write a new review directly inside the widget (no leaving). Writing a review auto-earns points via the "Write a review" earning rule. Reviews render in the shop's branding, data pulled live from Judge.me.
- Setup: Integration > Judge.me (connect first) → Unified Widget → Extensions → Reviews → choose Judge.me.

## Store Credit across storefront
Store Credit **balance + earn rate** now displays everywhere on the storefront, not just at checkout:
- Locations: widget **Balance card**, loyalty landing page, account pages, point calculator, thank you page, and reward listings (when reward type = Store Credit).
- **No separate toggle** — shows automatically when the shop has granted Store Credit permission AND has ≥1 active Store Credit program.
- Displayed in the shop's currency.
- In the Unified Widget, editable via the **Balance card**: choose design style, show/hide store credit, etc.
