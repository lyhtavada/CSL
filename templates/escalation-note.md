# Escalation Note Template

Use this format when creating Trello cards or Slack messages for escalation.

---

## Bug Escalation (to TS)

```
**Store:** [mystore.myshopify.com]
**Plan:** [Free / Basic / Pro / Plus / Professional / Advanced / Enterprise]
**Priority:** [P0 / P1 / P2 / P3]
**Product:** [Chatty / Joy Loyalty]

**Issue:**
[Brief description of the bug]

**Steps to Reproduce:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Expected:** [What should happen]
**Actual:** [What actually happens]

**Evidence:** [Screenshots / video links]
**Browser/Device:** [If UI-related]
**CS Troubleshooting Done:** [What you already checked/tried]
```

## Refund Escalation (to CS Leader)

```
**Store:** [mystore.myshopify.com]
**Plan:** [Current plan]
**Product:** [Chatty / Joy Loyalty]

**Request:** Refund
**Reason:** [Why the merchant wants a refund]
**Amount:** [Charge amount from invoice]
**Invoice Period:** [Billing cycle dates]
**Paid:** [Yes / No]
**Downgraded to Free:** [Yes / No]

**Context:** [Brief summary of the conversation — what was tried, merchant's sentiment]
```

## VIP / Angry Merchant (to CS Leader)

```
**Store:** [mystore.myshopify.com]
**Plan:** [Current plan]
**Product:** [Chatty / Joy Loyalty]
**VIP:** [Yes / No]

**Situation:** [Brief summary]
**Attempts Made:** [What CS has tried so far]
**Merchant Sentiment:** [Still angry / threatening review / threatening legal action]
**Recommended Action:** [Your suggestion for CS Leader]
```

## Feature Request (to PM)

```
**Requested by:** [Number of merchants who asked for this]
**Product:** [Chatty / Joy Loyalty]

**Feature:** [What they want]
**Use Case:** [Why they need it — what problem does it solve]
**Current Workaround:** [If any]
**Merchant Quotes:** [Direct quotes if available]
```
