## Notes for CS Team

- **Metafield support for metaobjects**: Feature in development — log all requests in tracking sheet.
- **Bulk conversation export**: TS can manually trigger; feature request for self-serve export is logged.
- **WhatsApp outbound messaging**: Not supported — Chatty can only respond to inbound WhatsApp messages.
- **Real-time inbox translation**: Pro+ plan only — inform merchants on Basic who ask.
- **Auto-transfer timer** (transfer to human after X minutes): Not a built-in feature — feature request noted.
- **GCLID / Google Ads offline conversion tracking**: Not currently supported — feature request noted.
- **Zendesk integration**: Supported, but only one Zendesk account per Chatty store (1:1). Merchants with 2 stores cannot share 1 Zendesk account.
- **Klaviyo integration**: Supported for contact sync. For outbound WhatsApp via Klaviyo — not currently supported.
- **Shopify Flow / API triggers**: Available — share documentation when asked. API docs at `https://help.chatty.net`.
- **Discount approvals**: NEVER approve without CSL. Escalate with full context.
- **Refund approvals**: NEVER approve without CSL. Collect invoice screenshot, reason, and escalate.
